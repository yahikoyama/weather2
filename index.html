This website provides automatically updated discomfort index (DI) data for Japan by using OpenWeatherMap API. The discomfort index is calculated using temperature, humidity, and weather conditions, offering a clear indicator of how hot or humid the environment feels.

The data is refreshed every 4 hours throughout the day, ensuring that the latest weather information is always available. This site is designed to support daily weather monitoring, heat countermeasures, and environmental observation.

All data is processed and published through an automated workflow using GitHub Pages. The goal of this project is to deliver simple, fast, and reliable weather insights without requiring any special applications or accounts.

For Studying Database System SQL and Programing by using weather data
This system is Client/Server System using:

Windows11 pro japanese (task scheduler)
PowerShell 5.1
Microsoft SQLServer Express 2022 (max size 10GB)
Python3.14 + pyodbc ODBC

weather.zip contains the source code and the database backup.
Please configure the conf file according to your environment and import the task scheduler XML file.
