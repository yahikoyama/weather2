About This Project

This website provides automatically generated daily weather reports in multiple languages. All reports are created using a fully automated workflow that collects meteorological data, processes it, and publishes formatted HTML pages to GitHub Pages.

What This Site Offers

  • Daily weather reports
  • Weekly summaries
  • Historical archives
  • Multilingual support (English, Russian, Chinese, Korean)

How It Works

A PowerShell automation script retrieves weather data, generates HTML files, and uploads them to GitHub using the GitHub API. The project also includes a custom multilingual dictionary system that enables consistent translations across all generated pages.

Project Structure

  • /daily/ — Daily weather reports 4h Update(English, Japanese,Russian, Chinese, Korean)
  • /historical/ — Weekly summaries(Japanese)
