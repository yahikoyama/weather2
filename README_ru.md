# Weather Report (GitHub Pages)
https://yahikoyama.github.io/weather2/

Этот проект предназначен для изучения  
**систем баз данных (SQL) и программирования**  
с использованием глобальных метеорологических данных.

Система получает данные о температуре, влажности и погодных условиях  
через API OpenWeatherMap и автоматически вычисляет  
**индекс дискомфорта (Discomfort Index: DI)**.  
Это позволяет определить самые прохладные регионы мира  
и предоставляет полезную информацию для борьбы с глобальным потеплением.

Если вы хотите добавить новые города,  
пожалуйста, внесите записи в таблицу **CityMaster**.

---

## ■ Обзор системы

Система построена по архитектуре Client/Server и использует:

- Windows 11 Pro (Планировщик задач)
- PowerShell 5.1
- Microsoft SQL Server Express 2022 (максимум 10 ГБ)
- Python 3.14 (с драйвером ODBC pyodbc)

Файл **weather.zip** содержит:
- полный исходный код  
- резервную копию базы данных  

Настройте файл **conf** в соответствии с вашей средой  
и импортируйте **XML-файл Планировщика задач**.
